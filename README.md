This is a mobile app for fun. App uses [joke API](https://api.chucknorris.io/) and generate you some Chuck Norris jokes.

## Screenshot:
<img src="assets/chuck-demo.jpeg" width ="250"/>

Currently includes:

- React Native
- React Navigation
- MobX State Tree
- TypeScript
- And more!

## Game logic:
Click the "Humor Me!" button and get some jokes.
